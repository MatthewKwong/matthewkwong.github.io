Figma Spotify Plugin

CDM 198 Final Project

April 2020 - Current

Background

This quarter as the final project for class, I wanted to build a Spotify player for the Figma design application. Unfortunately, the Spotify SDK was not availible to use on Figma. Given the limited amount of time left for the quarter, it was best for me to continue the intentions of making a music player on a webpage instead.

Timeline

When first starting this project, I identified the needs for the project to get a better sense of what I would need to accomplish by the end of the quarter.

Challenges

The biggest challenges for me was learning how to use the Spotify API and how to receive data back from API calls. This was an awesome learning journey and I'm so glad that I was able to build this product.

Revisions

Given more time, I would want to figure out a way to position the playlist name under the playlist image. Additionally, it would be nice to have the time stamp that could visually display the user's progress of a song.
